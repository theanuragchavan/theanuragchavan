## Featured Projects

### 🤖 [Behaviour-Aware HRI Controller](https://github.com/theanuragchavan/behaviour-aware-hri)
Adaptive robot storytelling assistant that infers a user's cognitive state (CONFUSED / THINKING / ENGAGED) from response latency and turn count, then adjusts hint level, tone, and repair strategy in real time via a rule-based controller fed to an LLM (GPT-4). Built with ROS / rospy, OpenAI API, and a tkinter voice interface. Logs every interaction to structured JSONL for downstream analysis.

`Python` · `ROS` · `OpenAI API` · `JSONL` · `pyproject.toml`

### 📡 [Big Data GPS Clustering — GeoLife](https://github.com/theanuragchavan/big-data-gps-analysis)
Distributed clustering of 11 million GPS points (Microsoft GeoLife dataset) on Databricks / PySpark. Custom MapPartitions-based KMeans, DBSCAN with Haversine feature engineering, spatial partitioning by lat/lon bucket, and a 200-run grid search. Best silhouette: KMeans 0.705 (cycling), DBSCAN 0.695 (cycling).

`PySpark` · `Databricks` · `KMeans` · `DBSCAN` · `Haversine`

---

### 📈 [BSE Algorithmic Trading Agent](https://github.com/theanuragchavan/bse-trading-agent)
Implemented the **Adaptive Aggressive (AA)** trading algorithm in the Bristol Stock Exchange simulator. EMA equilibrium tracking, Smith's Alpha volatility measurement, Newton-Raphson theta adaptation, and aggressiveness feedback rules. Ran 20 controlled experiments (Baseline ±5 vs High-Noise ±25) across 7 trader types; custom PT2 strategy achieved +336.57 cumulative profit and 4.26 Sharpe ratio.

`Python` · `BSE` · `Algorithmic Trading` · `Newton-Raphson` · `EMA`

### ✈️ [British Airways Data Science](https://github.com/theanuragchavan/british-airways-data-science)
Two-part industry project: (1) web-scraped and sentiment-analysed 1,000+ customer reviews using BeautifulSoup + VADER; (2) predicted flight booking completion with a Random Forest classifier (82% accuracy). Identified key booking drivers: purchase lead time, flight duration, route.

`Python` · `BeautifulSoup` · `Scikit-learn` · `VADER` · `Random Forest`
